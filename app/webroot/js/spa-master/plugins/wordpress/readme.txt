The purpose of this module is to convert  your website into a Single Page Application (SPA). It forms a base module for other real-time applications.

== Description ==
This module in itself won\'t appear to do all that much for you, but it is a requirement for some other real-time modules. Please be aware this module is still being developed and may well cause some features on your site to cease working as intended. If this happens, please deactivate the module and report the issue to the developers. The module is not active in the control panel so won\'t interfere with the module maintenance code.

== Installation ==
Simply install the module - there are currently no settings to tweak.

== Frequently Asked Questions ==
= How do I know if the module is working? =

* Install the module, then browse to your home page.
* Open the developer tools section in your browser and select \'console\'
* enter; fred=\"hello world\"
* now click on a link that is internal to your site, i.e. another of your pages
* enter; fred
* if it responds with \"undefined\", it\'s not working, if it responds with \"hello world\" it *is* working
